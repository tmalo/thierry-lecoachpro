# Checklist qualité offre

Score qualite = nombre de critères validés × 10 / 17 (arrondi à l'entier).

---

## Alignement marketing (5 critères)

| Clé                  | Critère                                                                 |
| -------------------- | ----------------------------------------------------------------------- |
| `zero_jargon`        | Zéro jargon coach/corporate dans tout le texte                          |
| `langage_client`     | Formulé dans le langage du client, pas celui du coach                   |
| `legitimite_tech`    | La légitimité tech (20 ans, CTO, archi, sécu) est visible              |
| `ton_direct`         | Ton direct et concret, pas marketé ni lissé                             |
| `posture_rogerienne` | Posture rogérienne : accompagnement de l'émergence, pas prescription   |

## Clarté (6 critères)

| Clé                      | Critère                                                            |
| ------------------------ | ------------------------------------------------------------------ |
| `cible_evidente`         | Un non-initié comprend immédiatement pour qui c'est                |
| `blocage_precis`         | Le blocage client est décrit précisément (situationnel, pas vague) |
| `transformation_concrete`| La transformation est concrète, mesurable et désirable             |
| `format_explicite`       | Format, durée et rythme sont explicites                            |
| `prix_affiche`           | Le prix est affiché clairement (montant exact ou fourchette)       |
| `processus_demarrage`    | Le processus pour démarrer est évident (CTA, premier pas)          |

## Différenciation (3 critères)

| Clé                 | Critère                                                              |
| ------------------- | -------------------------------------------------------------------- |
| `element_different` | Au moins un élément fait dire "c'est différent"                      |
| `double_legitimite` | La double légitimité tech + accompagnement est perceptible           |
| `parcours_original` | Le parcours ou la méthode est original ou surprenant                 |

## Cohérence portefeuille (3 critères)

| Clé                  | Critère                                                             |
| -------------------- | ------------------------------------------------------------------- |
| `pas_cannibalisation`| L'offre ne cannibalise pas une offre existante                      |
| `parcours_logique`   | S'inscrit dans un parcours logique avec les autres offres           |
| `cible_distincte`    | La cible est clairement distincte ou complémentaire                 |
